In questa cartella sono presenti 4 Test Suite elaborati con Selenium IDE come estensione del browser Google Chrome.

Prima di aprire i progetti nel proprio Selenium IDE cliccare su:
1) Estensioni -> Gestisci estensioni -> Selenium IDE "Dettagli" -> Spuntare "Consenti l'accesso agli URL dei file."
2) Eseguire la CreateDB.sql PRIMA di ripopolare il DB e solo dopo avviare le Test Suite.
